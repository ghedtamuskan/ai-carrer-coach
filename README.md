# AI Career Platform

An AI-powered job preparation platform that generates:
- Cover letters
- Interview questions
- AI feedback

Built with:
- Next.js
- ShadcnUi
- Prisma
- Clerk Authentication
- Gemini API (Demo Mode Supported)
- TailwindCSS

## Features
- Authentication
- Dynamic routing
- AI generation
- Server actions
- Protected pages

## Demo Mode
The project supports demo mode for cost-free AI simulation.
